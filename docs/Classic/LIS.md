## 使い方  
- `vector<int64_t> LIS(vector<int64_t>& v, bool non_strict = false, bool decrease = false)`：配列vの最長増加部分列を返す。ただし、non_strict = trueのとき単調性が広義に、decrease = trueのとき最長減少部分列になる。
